🍕 SQL Pizza Sales Analysis

## 📌 Project Overview
This project analyzes pizza sales data using SQL to generate business insights. It answers real-world business questions related to revenue, sales trends, customer ordering patterns, and best-selling pizzas.

## 🛠️ Tools Used
- SQL
- MySQL
- Excel

## 📊 Skills Demonstrated
- Joins
- Aggregate Functions
- GROUP BY & ORDER BY
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

## 📈 Business Questions Answered
- Total Revenue
- Average Order Value
- Total Pizzas Sold
- Top 5 Best-Selling Pizzas
- Revenue by Pizza Category
- Revenue by Pizza Size
- Monthly Sales Trend
- Peak Order Hours
